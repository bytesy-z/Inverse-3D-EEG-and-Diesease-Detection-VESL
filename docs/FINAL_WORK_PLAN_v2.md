# PhysDeepSIF — Final Work Plan v2

## 1. Project Summary

PhysDeepSIF is a physics-informed deep learning EEG inverse solver that reconstructs 76-region brain source activity from 19-channel scalp EEG. The model is trained entirely on synthetic data generated by The Virtual Brain (TVB) Epileptor simulations: each training sample satisfies the physical forward model `EEG = L @ S` (where L is the 19×76 leadfield matrix), allowing the neural network to learn the inverse mapping `S = f(EEG)` from 80,000 physics-compliant examples. The system provides a full clinical pipeline: EDF upload → source imaging → epileptogenicity biomarker detection → biophysical CMA-ES concordance analysis → XAI attribution — served through a Next.js web interface.

## 5. System Architecture

```
EDF upload (19ch, 200Hz)
    │
    ▼
Preprocessing: MNE load → channel mapping → bandpass 0.5–70Hz → notch 50Hz
→ segment 2s windows (400 samples) → per-channel de-mean → z-score normalize
    │
    ▼
PhysDeepSIF Inference: SpatialModule (19→128→256→256→128→76) per timestep
+ BiLSTM (hidden=76) → source activity (76 regions × 400 samples)  [~100ms]
    │
    ├── Phase A — Biomarker Detection  [~1s]
    │     AC variance → z-score → sigmoid → EI ∈ [0,1] → Top-10 regions
    │     "Preliminary Analysis" badge
    │
    └── Phase B — CMA-ES Concordance  [~7min, auto-started]
          CMA-ES proposes candidate x₀ (76-dim) → TVB Epileptor sim →
          EEG_sim = L @ S → score J(x₀) = MSE(PSD_sim, PSD_patient)
          + 0.1·||x₀+2.2||² → fitted x₀ → sigmoid → biophysical EI
          → concordance with Phase A (overlap ≥5/10 HIGH, 2–4 MODERATE, ≤1 LOW)
    │
    ▼
XAI Occlusion: Mask channels/time segments → rerun → channel + time importance
    │
    ▼
Frontend: WebSocket progress → concordance badge → updated heatmap
```

## 7. Quick Start

```bash
# Start both servers (runs dependency checks first)
./start.sh

# Start individually
./start.sh --backend   # Backend only (port 8000)
./start.sh --frontend  # Frontend only (port 3000)

# Verify health
curl -sS http://127.0.0.1:8000/api/health

# Run tests
pytest tests/ -m "not slow" -v

# Docker deployment
cd deploy && docker compose up --build
```
